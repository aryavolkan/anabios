All tasks complete. Evidence:

- **Substrate iterations (all observed in real runs):**
  - **Species-churn:** raw species-pair keys fragmented every feud across speciation splinters — keyed by lineage-faction root instead (same lesson as E3's guild series).
  - **Founder-root collapse:** archetype founders all carry placeholder parent `Some(0)`, so walking to root 0 merged every faction — faction root now stops below 0 (E5's LCA logic made the same distinction).
  - **Hits vs deaths:** combat *deaths* between factions are too rare to measure (max observed one-way score 10.9 < 12; mutual ~0) — cross-faction **hits** now feed hostility at 0.1 (wars are fought with volleys, deaths are decisive moments).
  - **Bidirectional over-engineering:** requiring two-way violence made war unreachable (prey never fights back in current worlds). Reverted to the event's own name: War**or Raid** — a sustained one-sided campaign is a raid. Clean hierarchy: Predation (first kill) → CombatRaid (burst) → WarOrRaid (sustained campaign).
- **Gate-gated mutation pool:** `SenseHostility` joins `random_node` only behind `war_enabled` — the ungated pool widening (weapons-style deliberate drift) broke `armed_lineages_persist_through_speciation`, which is what motivated the gate. Baselines byte-identical with the flag off.
- **Tests:** 4 war unit tests (declare/end, same-faction negative, trickle negative, normalization) + `tests/war.rs` integration (14 s debug).
- **Determinism:** `FORMAT_VERSION` 12→13; goldens regenerated once (golden scenarios have no cross-faction combat — hashes moved for layout only).
